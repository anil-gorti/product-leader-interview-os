# Examples

These files are filled examples of the templates — anonymised but structurally
complete. They show what a strong file looks like before you start filling
in your own content.

**proof-points-filled.md** — Shows the metric depth and context required.
Notice that every metric has the decision that preceded it, not just the outcome.

**career-arc-filled.md** — Shows the thesis structure. Notice that the
"What I Now Believe" section is the non-generic part — it takes 20 minutes
to write and is the most important section in the whole file.

**interviewer-card-filled.md** — Shows the difference between a LinkedIn
summary and an actual interviewer analysis. The "Real Test" section is
where the preparation work happens.

Use these as calibration, not as copy. Your files should be specific
to your career. If your proof-points file looks like this example,
you have understood the structure. If it says the same things,
you have filled in the wrong content.
